                PROBLEM                                                                 WORKING WEB APP

                Track chosen and problem tackled                                        What we built
    We chose Track 2: Food Security Delivery Operations. We                  We built Mealflo, a web application for food support delivery
    tackled the coordination problem faced by volunteer-led food             operations. It connects public intake, admin coordination, and
    delivery programs, where incoming food requests, volunteer               a volunteer driver experience, backed by a database layer,
    availability, meal inventory, routes, client needs, allergies, cold-     route calculation, map-based coordination, and language-
    chain requirements, accessibility constraints, and delivery              model-assisted processing for intake and inventory workflows.
    status all have to be managed at the same time.

    Our focus was helping a frontline coordinator turn scattered
    intake, inventory, and volunteer information into safe, assigned
    delivery routes with less manual work and fewer missed details
    - supported by a guided delivery system for volunteer drivers.
